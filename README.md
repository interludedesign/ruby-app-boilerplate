# Ruby App Boilerplate
A Ruby app boilerplate using RSpec, Zeitwerk, and Yard

## IRB Console
Start an IRB console from the command line
```sh
$ bin/console
```

## Run Specs
```sh
$ bundle exec rspec
```

## Generate App Docs
```sh
$ bundle exec yardoc
```

## Usage
Clone this repo and use it as a starting point for your app
